Djar is a minimal WordPress boilerplate starter theme. This  minimalist theme possible for designers and developers to use as a base to build websites for clients or to build completely custom themes from scratch.

For support and suggestions, go to: https://github.com/riipandi/djar/issues.

If you're creating your own theme or client project, open up all files and do a "Find and Replace All in All Files" on the word `djar` with your own slug and prefix.

Good luck!
